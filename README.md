## ⚙️ Особенности для Godot 4.6

- **Совместимость:** Проект создан под версию **4.6**, использует новый движок рендеринга.
- **Тип рендера:** По умолчанию настроен Vulkan (Forward+) с возможностью переключения на Compatibility (OpenGL 3).
- **Настройки ввода:** В `Project Settings` уже задан базовый список действий (ui_accept, ui_cancel, move_left, move_right, jump). При необходимости просто дополняйте.
- **Автозагрузка:** Папка `game/scripts/` предназначена для синглтонов (`Global.gd`, `AudioManager.gd`). Добавь их через Project Settings > Autoload.

## 🚀 Быстрый старт

1.  Открой папку проекта в **Godot 4.6** через кнопку «Импорт» или «Сканирование» в менеджере проектов. Если проект уже есть в списке, нажми «Редактировать».
2.  Основная игровая сцена подхватится автоматически из файла `project.godot` (поле `Main Scene`). Пока она не назначена, Godot предложит выбрать главную сцену при запуске.
3.  Создай первую сцену, например, «Уровень»:
    *   Щелкни правой кнопкой по папке `game/scenes` → **Создать новый** → **Сцена**.
    *   Добавь корневой узел (Node2D или Node3D) и сохрани как `level_01.tscn`.
    *   Зайди в **Проект → Параметры проекта → Общие → Запуск → Главная сцена** и укажи путь до `level_01.tscn`.
4.  Начни программировать! Добавляй спрайты и скрипты в соответствии с твоим дизайн-документом.

## 💡 Назначение папок

| Папка          | Зачем нужна                                                                 |
|----------------|-----------------------------------------------------------------------------|
| `assets/`      | Хранилище оригиналов. Перед использованием файлы импортируются в `resources/`. |
| `game/`        | Вся игровая логика, сцены уровней и объекты.                               |
| `ui/`          | Элементы интерфейса. Отделена для удобной замены скина игры.                |
| `resources/`   | Обработанные ассеты, готовые к использованию в Godot (импорты, материалы).  |
| `addons/`      | Сторонние плагины. В чистом проекте пуста.                                  |
